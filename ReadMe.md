## NavNinja
Imagine opening a tab and then again navigating to some page, applying some filters.  
Now imagine doing this everday for multiple websites !!! tiresome!! Isn't it??  

NavNinja helps you here.

Just add your urls (with or without the filter) in the 'urls_to_open' file.

Example:    
url_dict = {     
"Google": "https://www.google.com",  
"GitHub": "https://www.github.com",  
  
}

#### Some other UseCases: 
**Job Search:** save urls with your desired filters and positions, and location. 

**Track product(s**): Checking out a product or a range of products for a drop in price ? Add the url here.  
Example: ( https://www.amazon.de/s?k=samsung+s25&rh=n%3A562066%2Cp_n_feature_thirty-four_browse-bin%3A81335050031&s=price-asc-rank&dc&ds=v1%3AYFje5gv8B2qdxMAouRp3cKLjp1OyRwGP4EMjMkHAO9Y&crid=1YY5LZTJWLHU7&qid=1746008256&rnid=44358855031&sprefix=samsung+s25%2Caps%2C106&ref=sr_nr_p_n_feature_thirty-four_browse-bin_3 )
filters: Samsung S25, 256 GB, sort: low to high   


Add a shortcut of Nav_Ninja.bat to the desktop and Enjoy 😊 😎    
Save your time ⏳
