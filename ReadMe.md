## NavNinja
Imagine the hassle of opening tabs, navigating pages, and applying filters every day for multiple websites. 
!!! Tiresome 😩!! Isn't it??  

NavNinja is a tool designed to streamline this process, allowing you to open multiple websites (with filters) with just one click

![Alt-Text](assests/Navninja.png)  


#### Instructions:
Step 1: Open the 'urls_to_open' file.  
Step 2: Add your URLs (with or without filters) in the following format:  
  
    url_dict = {     
    "Google": "https://www.google.com",  
    "GitHub": "https://www.github.com",  
    
    }

Step 3: Save the file

Add a shortcut of Nav_Ninja.bat to the desktop and Enjoy 😊 😎    
Save your time ⏳
