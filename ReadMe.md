NavNinja
Imagine opening a tab and then again navigating to some page, applying some filters. Now imagine doing this everday for multiple websites !!! tiresome!! Isn't it??

NavNinja helps you here.

Just add your urls (with or without the filter) in the 'urls_to_open' file.

Example: url_dict = { "Google": "https://www.google.com", "GitHub": "https://www.github.com",

}

Add a shortcut of Nav_Ninja.bat to the desktop and double click and run it.
